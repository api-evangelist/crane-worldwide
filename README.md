# Crane Worldwide Logistics (crane-worldwide)

Crane Worldwide Logistics is a global third-party logistics provider delivering air freight, ocean freight, customs brokerage, warehousing, trucking, and supply chain solutions across more than 30 countries. Crane exposes a developer API portal powered by Tyk that gives partners and customers programmatic access to shipment, booking, and tracking data across the company's supply chain operations.

## APIs

### Crane Worldwide Logistics API

The Crane Worldwide Logistics API is a REST API that lets partners and customers connect their systems directly to Crane's logistics platform. Through a single approved API key, developers can request shipment bookings, retrieve tracking and milestone events, query shipment documentation, and integrate logistics data with their own ERP, OMS, and TMS systems. The portal supports OpenAPI specifications, JSON responses, and provides both sandbox and production environments.
